How computers store, access, and understand information, from the smallest bit to complex codes.

Primary Memory (Main Memory)

Volatile memory that holds data the CPU is actively using. Its contents are lost when power is off.

Cache Memory

An extremely fast buffer between the CPU and RAM. Uses SRAM technology and has multiple levels (L1, L2, L3).

RAM (Random Access Memory)

The main 'working' memory. Types include faster SRAM and common, cheaper DRAM which needs refreshing.

ROM (Read-Only Memory)

Non-volatile memory holding startup instructions (BIOS). Types: PROM, EPROM, EEPROM.

CMOS

Stores system configuration settings like date, time, and hardware info, powered by a small battery.

Secondary Memory (Storage)

Non-volatile memory for long-term data storage. It's slower but has a much larger capacity.

Magnetic Storage

Uses magnetism to store data. Includes Hard Disk Drives (HDD), Magnetic Tape, and older Floppy Disks.

Optical Storage

Uses lasers to read/write data. Includes CDs (~700MB), DVDs (~4.7GB+), and Blu-ray Discs (25GB+).

Flash / Electronic Storage

Uses electronic circuits with no moving parts. Includes fast SSDs, portable Pen Drives, and SD Cards.

Data Representation

How computers translate complex information into the simple language of binary numbers.

Memory Units

  • Bit: Smallest unit (0 or 1)
  • Nibble: 4 bits
  • Byte: 8 bits
  • Kilobyte (KB): 1024 Bytes
  • Megabyte (MB): 1024 KB
  • Gigabyte (GB): 1024 MB

Number Systems & Codes

  • Binary (Base 2): Uses 0 and 1.
  • Octal (Base 8): Uses 0-7.
  • Hexadecimal (Base 16): Uses 0-9 and A-F.
  • ASCII: 8-bit code for characters.
  • Unicode: 16/32-bit code for all world languages.
