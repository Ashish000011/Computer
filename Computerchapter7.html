Hardware & Software

The fundamental building blocks of any computer system: the physical components you can touch and the digital instructions that bring them to life.

Part 1: Hardware

The physical, tangible components of a computer system, from the main case to the smallest port.

The System Unit

UPS

An Uninterruptible Power Supply provides backup battery power during outages.

SMPS

A Switched-Mode Power Supply converts AC wall power to the DC power used by computer components.

Motherboard

The main circuit board (PCB) that connects all components, allowing them to communicate.

Computer Ports

Connecting sockets on the system unit for plugging in peripheral devices.

Serial Port

Older port (RS-232) that transmits data 1 bit at a time.

Parallel Port

Faster than serial, transmits 8 bits at once. Used for printers.

USB

Universal Serial Bus, the modern standard for connecting up to 127 devices.

FireWire

High-speed port (IEEE 1394) for large data transfers, like video.

VGA Port

An older, 15-pin analog port for connecting monitors.

HDMI & DVI

Modern digital ports for display devices. HDMI carries both video and audio.

SATA & PATA

Interfaces for connecting internal storage drives like HDDs and SSDs.

Other Ports

Includes Game Port for joysticks, Bluetooth for wireless, and MIDI for musical instruments.

The Computer Bus

A bus is a set of wires that acts as a data highway between computer components. This animation shows the three main types of system buses.

CPU
Memory
Address Bus: Carries memory addresses from the CPU to memory. It is unidirectional (one-way).
Data Bus: Carries the actual data back and forth between the CPU and memory. It is bidirectional.
Control Bus: Carries command and control signals from the CPU to manage all activities.

Part 2: Software

The set of instructions and programs that tell the hardware what to do. Software is intangible.

System Software

Core software that manages the hardware and provides a platform for applications.

  • Operating System (OS): The main system software (e.g., Windows, macOS).
  • Language Processor: Converts code into machine language (Compilers, Interpreters).
  • Utility Software: Helps maintain the computer (e.g., Antivirus, Disk Cleanup).

Application Software

Programs designed for end-users to perform specific tasks.

  • General Purpose: Word processors, web browsers, spreadsheets.
  • Specific Purpose: Accounting software, reservation systems.
  • Freeware: Software available at no cost.
  • Shareware: Try-before-you-buy software.
