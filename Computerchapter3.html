Organization & Input

Understanding the core structure of a computer, its brain (the CPU), and the tools we use to command it.

Fundamental Computer Organization

At its core, a computer's operation is a cycle of receiving input, processing it, and producing output. This is managed by the CPU.

The Machine Cycle (IPO)

The four fundamental steps the CPU takes to perform any task.

1Fetch: Retrieve data/instructions from RAM.
2Decode: Translate instructions for the CPU.
3Execute: Perform the actual command/calculation.
4Store: Save the result back to memory.

The Central Processing Unit (CPU)

The "brain" of the computer, composed of three main parts.

Control Unit (CU)

The coordinator. Manages the flow of data and tells other components what to do.

Arithmetic & Logic Unit (ALU)

The calculator. Performs all mathematical (+, -, *, /) and logical (AND, OR, >,<) operations.

Memory Unit (MU) / Registers

High-speed temporary storage within the CPU for data it's currently working on.

Input Devices

The tools and peripherals we use to send data and commands to the computer.

Keyboard

A primary data entry device with around 104 keys. Common layouts include QWERTY, AZERTY, and DVORAK.

Function Keys (F1-F12)

Perform special functions (e.g., F1 for Help, F5 for Refresh).

Modifier Keys

Modify other keys' actions (Ctrl, Alt, Shift).

Toggle Keys

Switch between modes (Caps Lock, Num Lock, Scroll Lock).

Navigation Keys

Move the cursor (Arrow Keys, Home, End, Page Up/Down).

Special Keys

Includes Esc, Delete, Backspace, Tab, etc.

Security Keys

Ctrl + Alt + Delete opens the Task Manager.

Pointing Devices

Control the cursor on the screen. Includes Mouse, Trackball, Joystick, Touchpad, and Light Pen.

Scanners

Convert physical documents/images into digital format. Types: Flatbed, Sheet-fed, Drum, Hand-held.

Readers

Read specific types of data. Includes OCR (text), OMR (marks), BCR (barcodes), MICR (cheques), MSR (cards).

Other Input Devices

Includes MIDI for musical instruments, Microphones for speech recognition, and Digitizers (Graphic Tablets) for drawing.
