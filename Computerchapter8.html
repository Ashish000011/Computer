Batch OS

The earliest type of OS. Processed jobs in batches without direct user interaction.

Real-Time OS (RTOS)

Responds to inputs instantly with no delay. Used in critical systems like robotics and medical devices.

Embedded OS

A specialized OS designed for a single purpose within a device, like in an ATM, car, or washing machine.

User Interfaces & GUI Elements

How we interact with the OS, from typing commands to clicking on graphical icons and controls.

CUI (Command-Line Interface)

A text-based interface where users type commands to interact with the OS. Powerful but less intuitive. Examples: MS-DOS, UNIX.

GUI (Graphical User Interface)

A visual interface using windows, icons, menus, and a pointer. Easy to learn and use. Examples: Windows, macOS.
