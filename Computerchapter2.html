Computer Types & Booting

From pocket-sized devices to room-sized giants, and the process that brings them all to life.

Classification by Size and Power

Computers range from personal devices to massive systems, each designed for specific tasks and user loads.

Microcomputer

Single-User

Also known as Personal Computers (PC). They are the most common type, designed for individual use. The first was the Altair 8800.

Examples: Desktop, Laptop, Smartphone, Workstation.

Minicomputer

~100 Users

More powerful than microcomputers, capable of supporting multiple users simultaneously. Often used as servers in small organizations.

Use Cases: Small businesses, factories, and schools.

Mainframe Computer

~1000s of Users

Large, powerful systems used by large organizations for critical applications that process massive amounts of data.

Use Cases: Banking, airline reservations, stock exchanges.

Supercomputer

Specialized Tasks

The fastest and most powerful computers, designed for complex scientific and engineering problems. Speed is measured in FLOPS.

Examples: CRAY-1 (first), FRONTIER (fastest), PARAM (Indian).

The Booting Process

Booting is the startup sequence that brings a computer to life, loading the operating system into memory so it can start working.

Hard Booting (Cold Boot)

Starting a computer from a completely powered-off state. It's the initial startup when you press the power button.

Soft Booting (Warm Boot)

Restarting a computer that is already on. This process is faster but risks losing unsaved data.

Booting Sequence

Power On & BIOS

When you press the power button, the CPU activates the BIOS (Basic Input/Output System), a program on the motherboard.

POST (Power-On Self-Test)

The BIOS runs a self-diagnostic test to check that all essential hardware components (RAM, keyboard, etc.) are working correctly.

Bootstrap Loader

The BIOS locates the Master Boot Record (MBR) on the hard drive and loads a small program called the Bootstrap Loader.

Operating System Loads

The Bootstrap Loader's job is to load the main kernel of the Operating System from the hard drive into the main memory (RAM). Once loaded, the OS takes control, and your computer is ready to use.
